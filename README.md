# Remnabot

Remnabot — это мощный телеграм-бот для управления панелью VLESS-узлов [Remnawave](https://github.com/remnawave/remnawave). Бот позволяет вам легко предоставлять доступ к VPN своим друзьям или клиентам прямо из Telegram, с удобным пользовательским интерфейсом **Админ-панели** прямо в мессенджере.

---

## 🌟 Основные функции

### 🧑‍💻 Для пользователей (ваших клиентов):
* **📊 Информация о подписке:** Быстрый просмотр трафика, даты истечения подписки и её статуса.
* **🔗 Получение ссылки:** Выдача конфигурационной ссылки (Subscription URL) для импорта в VPN-клиенты (V2rayNG, Shadowrocket, NekoBox и др.).
* **🛠 Поддержка:** Настраиваемое сообщение для быстрой связи.

### 🛡️ Для администратора:
* **👥 Управление базой:** Добавление пользователей, их активация и деактивация, удаление.
* **🕰 Управление временем:** Продление времени подписки (на дни, месяцы или навсегда).
* **🌐 Управление серверами:** Перезагрузка серверов, мониторинг онлайн пользователей. Поддерживает продвинутый мониторинг серверов через опциональную панель **Traffilk** (информация о CPU, RAM, Network).
* **🔄 Сброс трафика:** Ручной сброс использованного трафика для пользователей.
* **📱 Сброс привязок (HWID):** Просмотр и сброс привязанных к подписке устройств.

## 🚀 Установка (Быстрый старт)

Для установки вам потребуется сервер (VPS) с SSH. **Бот абсолютно автономен**, поэтому вы можете развернуть его как на том же сервере, где и панель Remnawave, так и на любом другом сервере через Docker.

### Шаг 1: Создание бота в Telegram
1. Перейдите к боту [@BotFather](https://t.me/BotFather) и отправьте команду `/newbot`.
2. Введите имя и юзернейм бота (юзернейм должен заканчиваться на `bot`).
3. Скопируйте **Токен** (он выглядит примерно так: `123456789:ABCdefGHIjklMNO...`).

### Шаг 2: Получение API-ключа от Remnawave
1. Зайдите в веб-интерфейс панели Remnawave.
2. Перейдите в меню **System -> API Tokens**.
3. Создайте новый ключ и **скопируйте его**.

### Шаг 3: Запуск на сервере
Подключитесь к вашему серверу по SSH и выполните следующие команды:

**1. Создайте папку для бота и скачайте нужные файлы:**
```bash
mkdir -p /opt/remnawave/remnabot && cd /opt/remnawave/remnabot
wget https://raw.githubusercontent.com/alexporteb/remnabot/main/docker-compose.yml
wget https://raw.githubusercontent.com/alexporteb/remnabot/main/.env.example -O .env
```

**2. Настройте файл `.env`:**
```bash
nano .env
```
Заполните обязательные переменные:
* `BOT_TOKEN` — ваш токен бота из Шага 1.
* `REMNAWAVE_API_URL` — укажите URL вашей панели Remnawave (например, `http://localhost:3000` или `https://вашдомен`).
* `REMNAWAVE_X_API_KEY` — ваш API ключ из Шага 2.
* `ADMIN_TELEGRAM_IDS` — ваш Telegram ID (узнать его можно через бота [@getmyid_bot](https://t.me/getmyid_bot)).

*(Опционально)* Если вы используете панель мониторинга **Traffilk**, вы можете настроить интеграцию:
* `TRAFFILK_API_URL` — URL вашей панели Traffilk (например, `http://localhost:8080`)
* `TRAFFILK_API_KEY` — API ключ от Traffilk (формат `trfk_...`)

*Чтобы сохранить изменения в nano, нажмите `Ctrl+O`, затем `Enter`, затем `Ctrl+X`.*

**3. Запустите бота:**
```bash
docker compose pull
docker compose up -d
```

🎉 **Всё готово!** Перейдите в вашего бота в Telegram и отправьте команду `/start`.

---

## 🛠️ Админ-панель

Если ваш `Telegram ID` был указан в файле `.env`, вы автоматически получаете права **Администратора**.
Для открытия Админ-панели отправьте боту сообщение:
`/admin`

Здесь вы сможете управлять пользователями, серверами и конфигурацией бота!

---

## 🔄 Обновление бота

Когда выходит новая версия бота, вы можете обновить его за пару секунд. Выполните эти команды:
```bash
cd /opt/remnawave/remnabot
docker compose pull
docker compose up -d
```
*Все ваши настройки и база активных пользователей (в папке `data/`) сохранятся!*

---

## 🗑 Удаление бота
Если вы хотите полностью удалить бота со всеми его файлами и образами Docker, выполните:
```bash
cd /opt/remnawave/remnabot && docker compose down && cd .. && rm -rf remnabot && docker rmi ghcr.io/alexporteb/remnabot:main
```
