# 🤖 Remnabot — Умный Telegram бот для панели Remnawave

Remnabot — это мощное дополнение для вашей VLESS-панели [Remnawave](https://github.com/remnawave/remnawave). Бот позволяет вашим клиентам удобно управлять своими подписками через Telegram, а вам предоставляет полноценную **Админ-панель** прямо в мессенджере.

---

## ✨ Основные возможности

### 👤 Для пользователей (ваших клиентов):
* **📊 Просмотр профиля:** Проверка остатка трафика, даты истечения подписки и её статуса.
* **🔗 Управление подпиской:** Быстрое получение ссылки (Subscription URL) для автонастройки VPN-клиентов (V2rayNG, Shadowrocket, NekoBox и др.).
* **🔔 Напоминания:** Автоматические уведомления об окончании подписки.

### 👑 Для администратора:
* **👥 Управление базой:** Удобный просмотр всех клиентов с разделением по статусу (Активные, Истекшие, Отключенные).
* **⏳ Продление подписки:** Продление доступа на месяц, год или установка "Безлимита" в 2 клика.
* **🛑 Смена статусов:** Приостановка, возобновление или полное удаление пользователей.
* **📢 Массовые рассылки:** Отправка текста, картинок и "кружочков" всем вашим клиентам одновременно.
* **✉️ Личные сообщения:** Возможность отправить персональное уведомление конкретному пользователю.
* **⚙️ Настройка авто-уведомлений:** Настройка текста, дня и времени отправки автоматических напоминаний об оплате.

## 🛠 Установка (Быстрый старт)

Для установки вам потребуется только доступ к вашему серверу (VPS) по SSH. **Бот автоматически собран**, поэтому вам не нужно устанавливать никаких сложных зависимостей (только Docker).

### Шаг 1: Создайте бота в Telegram
1. Перейдите к боту [@BotFather](https://t.me/BotFather) и отправьте команду `/newbot`.
2. Следуйте инструкциям, чтобы задать имя и юзернейм вашему боту.
3. Скопируйте полученный **Токен** (он выглядит примерно так: `123456789:ABCdefGHIjklMNO...`).

### Шаг 2: Получите API-ключ от Remnawave
1. Зайдите в веб-интерфейс вашей панели Remnawave как администратор.
2. Перейдите в раздел **System -> API Tokens**.
3. Создайте новый токен и **скопируйте его**.

### Шаг 3: Запуск на сервере
Подключитесь к вашему серверу по SSH и выполните следующие команды:

**1. Создайте папку для бота и скачайте нужные файлы:**
```bash
mkdir -p /opt/remnawave/remnabot && cd /opt/remnawave/remnabot
wget https://raw.githubusercontent.com/alexporteb/remnabot/main/docker-compose.yml
wget https://raw.githubusercontent.com/alexporteb/remnabot/main/.env.example -O .env
```

**2. Настройте файл `.env`:**
```bash
nano .env
```
Заполните ваши данные внутри файла:
* `BOT_TOKEN` — вставьте токен из Шага 1.
* `REMNAWAVE_API_URL` — оставьте `http://remnawave:3000`, если бот установлен на том же сервере, что и панель.
* `REMNAWAVE_X_API_KEY` — вставьте токен из Шага 2.
* `ADMIN_TELEGRAM_IDS` — укажите ваш Telegram ID (узнать его можно у бота [@getmyid_bot](https://t.me/getmyid_bot)).

*Чтобы сохранить изменения в nano, нажмите `Ctrl+O`, затем `Enter`, затем `Ctrl+X`.*

**3. Запустите бота:**
```bash
docker compose pull
docker compose up -d
```

🎉 **Всё готово!** Перейдите в вашего бота в Telegram и отправьте команду `/start`.

## 👨‍💻 Как пользоваться админкой?

Если ваш `Telegram ID` был правильно указан в файле `.env`, вы можете вызвать **секретное меню администратора**.
Для этого просто отправьте боту сообщение:
`/admin`

Здесь вы найдете все инструменты для управления вашими клиентами, рассылками и настройками бота.

---

## 🔄 Как обновить бота?

Если вышла новая версия с новыми функциями, обновление займет всего 10 секунд. Выполните эти команды:
```bash
cd /opt/remnawave/remnabot
docker compose pull
docker compose up -d
```
*Важно: ваши настройки и база активных пользователей (папка `data/`) сохранятся при обновлении.*

---

## ❌ Полное удаление
Если вы решите удалить бота, просто выполните эту команду (данные пользователей в самой панели Remnawave затронуты не будут):
```bash
cd /opt/remnawave/remnabot && docker compose down && cd .. && rm -rf remnabot && docker rmi ghcr.io/alexporteb/remnabot:main
```
